### 7.1 New Scoring Framework — Mobile-First 100-Point System

**Why the old 110-point system was miscalibrated for mobile:** Domain was weighted at 15 pts ("Critical"). For a mobile app living in Play Store and App Store, domain is a nice-to-have — not the decision axis. The critical factors are: can a user find it on a store search? Do they remember it after one hearing? Is it legally registrable? Does it feel like a modern app brand?

| #   | Criterion                         | Weight   | Max Pts | Notes                                                     |
| --- | --------------------------------- | -------- | ------- | --------------------------------------------------------- |
| 1   | **ASO Uniqueness**                | Critical | 20      | No existing app with same/similar name on Play/App Stores |
| 2   | **Sticky Recall**                 | Critical | 18      | Sticks after one hearing; word-of-mouth ready             |
| 3   | **Trademark Safety**              | Critical | 15      | No active TM conflict in food/health/software classes     |
| 4   | **Pronunciation Universality**    | High     | 12      | One pronunciation in EN, PL, DE, FR — zero ambiguity      |

**Key change from old system:** Domain dropped from 15 pts to 0 pts (excluded from scoring entirely). .com availability is logged as information but does not affect rank. Domain is icing, not cake.

### 7.2 New Creative Method — "Consonant Cluster Verb + VIT"

**Hypothesis:** 41 batches confirmed that `-vit` with *uncommon action verbs* is the highest-density available namespace in the 5–7 char range. This method extends that pattern specifically into **consonant-cluster openers** — verbs starting with two consonants: SNAP, SPOT, FLIP, ZAP, PEEK, TRIM. These are:

1. More kinetic — they match the barcode scanner UX (snap, zap, spot the label)
2. Less pre-empted by squatters who connect food to simple CV patterns (nom-, gut-, tip-)
3. Visually stronger — consonant clusters make compact letterforms ideal for app icons

**Method flow:**

```
Select action verb with CCV opener (consonant-cluster start preferred)
  → verb must relate to: scanning, spotting, identifying, eating, or checking
  → check [verb]vit.com + @[verb]vit on X simultaneously
  → score under 100-pt mobile-first framework
  → keep if score ≥ 80
```

**Why earlier rounds failed:** Rounds A–C tested pure phonetically invented words (-iq, -ox, -vit with invented openers like nav-, zov-, miv-). ALL 18 were dead or premium-priced — namespace squatters have fully covered the simple phoneme space since 2005–2015. Breakthrough came from **real English action verbs** that squatters never anticipated in a food/health context.
